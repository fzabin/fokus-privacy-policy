# Política de Privacidade de Fokus

A privacidade dos nossos usuários é muito importante para nós. Esta Política de Privacidade descreve como tratamos as informações no aplicativo Fokus.

1. Coleta de Informações
O aplicativo Fokus foi desenvolvido como um serviço gratuito. Este serviço é fornecido sem custos e foi projetado para ser usado no estado em que se encontra. Nós NÃO coletamos, armazenamos, compartilhamos ou transmitimos qualquer tipo de informação pessoal, dados de uso, identificadores de dispositivos ou dados de localização dos usuários.

2. Anúncios e Serviços de Terceiros
O aplicativo NÃO exibe anúncios e NÃO utiliza serviços ou SDKs de terceiros que possam rastrear suas informações ou coletar dados sobre você.

3. Alterações a Esta Política de Privacidade
Podemos atualizar nossa Política de Privacidade de tempos em tempos. Portanto, é recomendável revisar esta página periodicamente para verificar quaisquer alterações. Avisaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.

4. Contato
Se você tiver alguma dúvida ou sugestão sobre nossa Política de Privacidade, não hesite em entrar em contato conosco pelo e-mail: fernando.zabin@gmail.com.

Esta política é de uso exclusivo para o aplicativo Fokus e entra em vigor a partir de 9 de junho de 2026.
